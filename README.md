## Deskripsi Project

Project ini merupakan tugas individu mata kuliah Pengolahan Sinyal Digital yang bertujuan untuk mempelajari manipulasi citra digital menggunakan Python dan OpenCV. Beberapa teknik yang diterapkan meliputi konversi warna, resize citra, image blending, image negative, transformasi logaritmik, dan transformasi gamma.

## Citra yang Digunakan

* Image 1 : Poster Thunderbolts
* Image 2 : Poster Spider-Man

## Library

* OpenCV
* NumPy
* Matplotlib

## Cara Menjalankan

1. Buka file notebook pada Google Colab atau Jupyter Notebook.
2. Hubungkan Google Drive.
3. Pastikan kedua citra berada pada folder yang sesuai.
4. Jalankan setiap cell secara berurutan.

## Ringkasan Hasil

Eksperimen menunjukkan bahwa setiap teknik manipulasi citra memiliki fungsi yang berbeda. Image blending digunakan untuk menggabungkan dua citra, image negative membalik nilai pixel, transformasi logaritmik meningkatkan detail area gelap, dan transformasi gamma digunakan untuk koreksi pencahayaan.

## Kesimpulan

Manipulasi citra digital menggunakan OpenCV dapat dilakukan dengan berbagai metode sesuai kebutuhan. Pemilihan teknik yang tepat dapat meningkatkan kualitas visual dan informasi yang terkandung dalam citra.
